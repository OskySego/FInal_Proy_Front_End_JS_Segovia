# README.md

## Propósito de la Página
Este proyecto es un e-commerce dinámico para Osky Diseños, especializado en servicios de diseño web y gráfico. Consume datos de una API REST para mostrar productos, permite agregar al carrito usando localStorage para persistencia, y incluye un formulario de contacto funcional con Formspree. El sitio es responsivo, accesible y optimizado para SEO básico.

## Estructura
- **index.html**: Página principal con bienvenida y resumen.
- **productos.html**: Lista de servicios cargados desde API.
- **resenas.html**: Reseñas de clientes.
- **contacto.html**: Formulario de contacto.
- **css/styles.css**: Estilos responsivos con Flexbox y Grid.
- **js/script.js**: Interactividad, fetch API, carrito y animaciones.

## Tecnologías
- HTML5 semántico.
- CSS3 con Flexbox, Grid y Media Queries.
- JavaScript con Fetch API y localStorage.
- GSAP para animaciones.
- Formspree para formulario.

## Instrucciones
1. Abre index.html en un navegador.
2. Explora las páginas.
3. Agrega productos al carrito; persiste al refrescar.
4. Envía el formulario de contacto.
